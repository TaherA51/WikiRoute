How It Works

WikiRoute uses advanced graph algorithms to find the shortest path between any two Wikipedia articles. Our algorithm prioritizes routes that go through more closely related articles by using weighted edges.

Algorithm Details:

  • Dijkstra's Algorithm: Finds the shortest path in weighted graphs
  • Edge Weighting: Bidirectional links have weight 1, unidirectional links have weight 2
  • API Integration: Uses MediaWiki API to fetch article links in real-time
